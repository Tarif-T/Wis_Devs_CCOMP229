# Wise Devs - COMP229 Final Project

Meetup-style MERN web application where users can register, log in, create events, discover events, and join community experiences.

## Team

- Mohammadali (Ali) Ghanavati - Back-End Developer
- Ying-Chieh Chen - Back-End Developer
- Mingun Choi - Developer
- Tarif Abdalla - Front-End and UI/UX Designer

## Frontend (Tarif Abdalla)

The full React UI/UX implementation is in [client](client), including:

- Responsive navigation and branded landing page
- Event discovery grid with filters UI
- Event details page
- Create-event form UX
- Login/Register screens
- Member dashboard view
- Reusable components and design system tokens

## Run Frontend

1. `cd client`
2. `npm install`
3. `npm run dev`

Build for production:

1. `cd client`
2. `npm run build`
